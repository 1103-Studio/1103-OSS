# 1103-OSS

高性能、S3 兼容的对象存储系统。

## 你能用它做什么

- 图片/视频存储与 CDN
- 应用静态资源托管
- 备份与归档存储
- 私有云对象存储方案

## 核心能力

- S3 API 兼容（支持 AWS SDK / AWS CLI / 常见第三方工具）
- AWS Signature V4 签名认证
- 大文件分片上传（Multipart Upload）
- Go 后端 + React 管理控制台
- Docker Compose 一键启动（开发 / 生产）
